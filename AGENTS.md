# AGENTS.md

Sistren — Sistem Informasi Terpadu (Highschool Information System)

## Rules

1. **bun-only** — Use `bun` for all package management and script execution. Do not use `npm`, `yarn`, or `pnpm`.
2. **no manual deps** — Do not manually edit `package.json` to add or remove dependencies. Use `bun add` or `bun remove` instead.
3. **@latest always** — Always use `@latest` when adding packages (e.g., `bun add package@latest`)
